Squad Dev, v3

Uma equipe de TI, em uma pasta.

Cole a Squad_Dev/ no seu projeto, configure a regra persistente da IA que você usa, e comece a desenvolver. O guia abaixo mostra só o que precisa ser configurado em cada ferramenta.

Regra persistente

Texto padrão, serve pra todas as IAs

O texto é o mesmo em todo lugar. O que muda é onde salvar.

System Prompt 
Você é a Squad Dev. Antes de qualquer ação:
1. Leia Squad_Dev/.agents/agents.md para identificar o agente certo.
2. Leia a skill em Squad_Dev/.agents/skills/<skill>/SKILL.md.
3. Leia Squad_Dev/production_artifacts/memory/AI_CONTEXT.md para retomar contexto.
4. Código em Squad_Dev/app_build/.
5. Documentação em Squad_Dev/production_artifacts/.
6. Security Gate é bloqueante antes de deploy.
Claude Code Antigravity Cursor Copilot Windsurf Claude Desktop Gemini ChatGPT

1. Claude Code Nativo

Skills auto-detectadas. Não precisa configurar nada.

  1. Cole a pasta

    Coloque Squad_Dev/ na raiz do projeto.

  2. Rode o CLI

    bash
    claude

  3. Invoque a Squad

    prompt
    /dev-squad Quero criar um app de gestão de tarefas para equipes

2. Google Antigravity Nativo

Antigravity lê .agents/ e segue os workflows nativamente. Os agentes em agents.md ficam disponíveis no chat, e os arquivos em workflows/ funcionam como Missions.

  1. Cole a pasta no workspace

    Coloque Squad_Dev/ na raiz do workspace e abra no Antigravity.

  2. Execute o pipeline completo

    prompt
    Execute Squad_Dev/.agents/workflows/startcycle.md para: [ideia].
Identifique os agentes em .agents/agents.md e siga as skills em .agents/skills/.

  3. Security Gate por Sprint

    prompt
    Execute Squad_Dev/.agents/workflows/sprint-security-gate.md no Sprint atual.
DicaAntigravity tem terminal, browser e filesystem por padrão. Missions longas funcionam bem com startcycle.md.

3. Cursor

  1. Cole a pasta

    Squad_Dev/ na raiz do projeto.

  2. Crie a rule

    Arquivo .cursor/rules/squad-dev.mdc:

    .cursor/rules/squad-dev.mdc
    ---
description: Squad Dev agents and workflow
globs: ["**/*"]
alwaysApply: true
---

6. Claude Desktop (MCP filesystem)

  1. Configure o MCP filesystem

    Aponte o servidor pra raiz do projeto que contém Squad_Dev/.

  2. Cole na thread

    Em cada nova conversa, cole o texto padrão seguido da tarefa.

    prompt
    [texto padrão da Squad Dev]

Aja como solution-architect e projete a arquitetura para: [ideia].

7. Gemini (CLI e AI Studio)

AI Studio (web)

8. ChatGPT (Projects)

  1. Crie um Project

  2. Upload dos arquivos

  3. Project Instructions

    Cole o texto padrão.

  4. Inicie

    prompt
    Ative /dev-squad para: [ideia]

Dicas universais

Memória viva

Ao fim de cada fase, peça: "Atualize production_artifacts/memory/AI_CONTEXT.md com stack, sprint atual, onde paramos e próximos passos."

Progressive disclosure

Não carregue todas as skills de uma vez. Carregue só a do agente ativo. Abra references/ quando precisar de detalhe.

Sessões curtas

Feche a sessão ao fim de cada fase, salve em AI_CONTEXT.md, abra nova sessão na próxima.

Security Gate antes de deploy

prompt
Execute Squad_Dev/.agents/workflows/sprint-security-gate.md no código atual
de app_build/ e entregue o Security_Gate_Report.md.
