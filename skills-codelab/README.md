# skills-codelab

`skills-codelab` é a célula experimental de desenvolvimento e testes da aplicação.

Mais do que uma pasta técnica, este espaço funciona como uma pequena equipe-orquestra: aqui vivem os agentes, skills, fluxos e materiais de apoio usados para pensar, construir, validar e evoluir features com mais velocidade e mais consistência.

## O que este espaço representa

Este diretório reúne a base operacional de uma equipe de desenvolvimento assistida por IA, voltada para:

- planejamento técnico
- arquitetura de solução
- geração de código
- revisão e auditoria
- QA e bugfix
- deploy e operação
- documentação
- testes de fluxos de entrega

## Visão da equipe

Pense no `skills-codelab` como um codelab vivo:

- o Product e a Arquitetura ajudam a definir o caminho
- os especialistas técnicos aceleram implementação
- os agentes de QA e auditoria ajudam a reduzir risco
- os fluxos e skills mantêm a operação organizada

É uma base feita para experimentar, iterar, validar e transformar ideias em entregas mais rápido.

## Estrutura principal

- `.agents/`
  Núcleo da equipe autônoma, com personas, workflows e skills especializadas.

- `app_build/`
  Área destinada a build, montagem ou estrutura operacional da aplicação durante o ciclo de desenvolvimento.

- `production_artifacts/`
  Espaço para artefatos, referências e materiais ligados à entrega e à versão de produção.

- `API_KEYS_SETUP.md`
  Guia de configuração de chaves e credenciais necessárias para o pipeline.

## Objetivo prático

Manter uma base organizada para que desenvolvimento, testes e operação não fiquem soltos.
Aqui, cada skill e cada agente ajudam a compor um pipeline mais inteligente para evolução contínua da aplicação.

## Observação

Este diretório pode conter estruturas de apoio, documentação operacional e componentes de automação que servem como laboratório da equipe de desenvolvimento e testes.
